# 🚢 Titanic Mini Visualization Dashboard

## 📌 Project Overview

This project was developed as part of the **Data Science with Python Internship – Task 4**. It presents a mini visualization dashboard built using the Titanic dataset to explore passenger survival patterns through data cleaning, feature engineering, and insightful visualizations.

The notebook demonstrates a complete Exploratory Data Analysis (EDA) workflow and highlights how visual storytelling can be used to uncover meaningful trends in real-world datasets.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Create meaningful features for analysis.
* Visualize important relationships using Matplotlib and Seaborn.
* Communicate findings through concise, insight-driven charts.
* Build a well-structured and reproducible data analysis project.

---

## 📂 Dataset

* **Dataset:** Titanic Dataset
* **Target Variable:** `Survived`
* **Records:** Passenger demographic and travel information from the Titanic disaster.

## 📊 Project Workflow

1. Import required libraries.
2. Load and inspect the dataset.
3. Clean the data by handling missing values and duplicates.
4. Perform feature engineering (`FamilySize` and `IsAlone`).
5. Create multiple visualizations.
6. Interpret each visualization with key insights.
7. Summarize the overall findings.

## 📈 Visualizations Included

* Survival Distribution
* Survival by Gender
* Passenger Class Distribution
* Survival Rate by Passenger Class
* Age Distribution
* Fare Distribution
* Correlation Heatmap
* Family Size vs Survival
* Survival by Embarkation Port
* Pairwise Relationship Analysis (Pairplot)

---

## 🔍 Key Insights

* Female passengers had a significantly higher survival rate than male passengers.
* First-class passengers were more likely to survive than passengers in lower classes.
* Most passengers were between 20 and 40 years old.
* Higher ticket fares generally corresponded to higher survival rates.
* Passengers travelling alone or in smaller families showed better survival outcomes.
* Feature engineering improved the understanding of passenger characteristics.

## 🚀 How to Use

1. Clone this repository or download the project files.

   ```bash

   ```
4. Open **Task4_titanic.ipynb** using Jupyter Notebook or Google Colab.
5. Run all cells from top to bottom.
6. Explore the visualizations and read the insights provided below each chart.
